How To Use ZevSafe

A simple, step-by-step guide to encrypting and decrypting your folders with military-grade AES-256-GCM security — all inside your browser.

🔐 Encrypting a Folder

1

Open ZevSafe

Open index.html in any modern browser (Chrome, Edge, Firefox, Safari). No installation needed. Works 100% offline after the page loads.

2

Select Your Folder

On the left panel, either drag & drop your entire folder onto the drop zone, or click "Browse Folder" to pick it from your file system. All files inside will be included automatically — even nested subfolders.

3

Set a Strong Password

Type a password in the Password field. Watch the strength meter — aim for at least Strong (mix uppercase, numbers, and symbols). Re-enter it in Confirm Password. Use the 👁 button to reveal what you typed.

4

Click "Encrypt & Download"

ZevSafe will compress your folder into a ZIP, derive an AES-256-GCM key from your password using PBKDF2, encrypt everything, and automatically download a single yourfolder.enc file. The progress bar and live log show each step.

5

Store Your .enc File Safely

Save the .enc file on your USB drive, SD card, or cloud storage. Remember your password — there is no recovery option. The file is useless without it.

🔓 Decrypting a Vault

1

Open ZevSafe

Open index.html in your browser. You can do this on any computer, phone, or tablet — no account or special software needed.

2

Load Your .enc File

On the right panel, drag & drop your .enc vault file, or click "Select .enc File". Make sure the file has not been renamed or modified — any changes will cause decryption to fail.

3

Enter Your Password

Type the exact same password you used when encrypting. AES-GCM automatically verifies authenticity — a wrong password will immediately show an error without decrypting anything.

4

Click "Decrypt & Download"

ZevSafe will verify your password, decrypt the vault, and download a yourfolder_decrypted.zip file containing all your original files with their folder structure preserved.

5

Extract the ZIP

Right-click the downloaded .zip file and choose Extract All (Windows) or double-click (macOS) to restore your original folder structure.

🛡️ How the Encryption Works

Parameter Value & Detail
Cipher AES-256-GCM (Authenticated Encryption)
Key Derivation PBKDF2 — SHA-256, 100,000 iterations
Salt 16 bytes — random per encryption (stored in file header)
IV / Nonce 12 bytes — random per encryption (stored in file header)
Authentication Tag 128-bit GCM tag — detects any tampering automatically
Compression DEFLATE level 6 (via JSZip) before encryption
Vault file format Salt (16B) + IV (12B) + Ciphertext + GCM Tag
Entropy source window.crypto.getRandomValues() — browser's CSPRNG
Crypto engine Web Crypto API — native browser, no external library
Network usage Zero — 100% offline after initial page load

💡 Tips & Warnings

✅ Use a Passphrase

A long passphrase (e.g. "blue-tiger-rain-42!") is far stronger than a short complex password. Aim for 16+ characters.

❌ No Password Recovery

There is absolutely no backdoor, reset, or recovery. Losing your password means losing your data permanently.

⚠️ Keep a Backup

Always keep the original files somewhere safe until you verify decryption works correctly. Test decrypt before deleting originals.

🔒 Offline is Safer

For maximum security, download ZevSafe and use it without internet. Your files and password never leave your RAM.

✅ Works on SD Cards

ZevSafe is perfect for encrypting folders on USB drives and SD cards. The .enc file is portable — open it anywhere.

❌ Don't Rename .enc

Renaming the .enc file is fine, but never modify its contents. Even 1 changed byte will cause decryption to fail.
